# About

Freshclass is an easy and quite simple way to style a new webpage/site. Freshclass works with blogs, personal sites, landing pages, and more! Freshclass's dynamic and sleek design will make your job of turning pages responsive into a simple task. To get started copy the code to our stylesheet and paste it beetween the <head> tags on your site: 


<link
  rel="stylesheet"
  href="https://freshclass.netlify.app/freshclass.min.css"
/>

How it works

But how does it work so well? It's a CSS Framework, which is a CSS file that gives you easy templates for designing an amazing site. But unlike other frameworks such as Bootstrap, Declass controls your site's look based on tags, not classnames. What does that mean? It means that instead of having to bombard your site with class attributes, you just write the HTML and we make it beautiful for you. Styling simplified. Because as the saying goes,

"Less is more."

Design

Freshclass has a simple & sleek design with readable and big fonts, lots of space, and distinct style of sharp but smooth edges. It also supports a --color variable on your page which gives your page a nice flair instead of just black and white. Declass's style also supports a accessibility mode wich runs on the users device when they select the option. If you want to learn more about accessibility mode, read our Documentation.

Tags

Just how does Freshclass give you so many options with so little tags? Does it create a new kind of attribute for styling? Does it add more tags? No. The answer is one word: Creativity. Declass lets you combine tags to make objects such as link buttons, centered headings, or preformatted code.

Almost all of the tags in HTML5 are supported by Freshclass. Knowing which tags to use to make a nice and responsive site are crucial, so make sure to check out our docs, which have easy-to-read information and easy-to-follow guides on how to create items such as dropdowns, footers, sidebars, navigation bars, and much more!
